Access Token 발급 방법

  1. Meta for Developers에서 앱 대시보드 접속
  2. 왼쪽 메뉴에서 "Threads API" → "Token Generator" 선택
  3. 권한에서 threads_basic 선택
  4. "Generate Access Token" 클릭
  5. 생성된 토큰을 복사하여 위에 입력

⏰ 토큰 만료 정보

  • Long-lived 토큰은 60일 후 만료됩니다
  • 만료 7일 전부터 갱신 알림이 표시됩니다
  • "토큰 갱신" 버튼으로 60일 연장할 수 있습니다
  • 토큰이 완전히 만료되면 새로 발급받아야 합니다
