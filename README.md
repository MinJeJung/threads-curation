# Threads Curation

Threads API를 사용한 개인 포스트 달력 큐레이션 사이트

## 🚀 빠른 시작

### 1. 토큰 설정

### 2. 기능

- 📅 **달력 뷰**: 월별로 포스트를 시각화
- 📊 **통계**: 이달의 포스트 수, 활동 일수, 최대 일일 포스트
- 🔄 **자동 토큰 갱신**: 만료 7일 전 알림 및 60일 연장 기능
- 📱 **반응형**: 모바일, 태블릿, 데스크톱 모두 지원

## 🔧 토큰 발급

1. [Meta for Developers](https://developers.facebook.com/apps) 접속
2. 앱 대시보드에서 "Threads API" → "Token Generator" 선택
3. `threads_basic` 권한 선택
4. "Generate Access Token" 클릭
5. 생성된 토큰 복사
